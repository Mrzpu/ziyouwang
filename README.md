# 自由网 · 精选酒店平台

> 不靠排名，不接竞价广告——只推真正值得住的酒店。

---

## 项目简介

**自由网**是一个精选酒店预订平台，起步于山东威海。

中国主流 OTA（携程、飞猪、美团）的酒店排名由广告竞价决定，排在前面的不一定是最好的。自由网的逻辑完全不同：

- **人工筛选**：每个城市只收录 5–10 家，每家经过人工评测
- **批发价格**：直接与酒店签订年度批量采购协议，价格比携程低 15–25%
- **不接竞价**：排名只看真实评分，酒店无法花钱买排名
- **会员制**：年费 ¥299，覆盖全部精选酒店的批发价格

## 商业模式

- **收入来源**：会员年费（¥299/年）+ 酒店入驻服务费
- **核心优势**：批量采购协议 → 价格比携程低 15–25%，用差价换用户
- **护城河**：精选筛选机制 + 不可买排名的信任背书
